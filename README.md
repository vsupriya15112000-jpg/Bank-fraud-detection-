# 🏦 Bank Fraud Detection System

## Overview

This project is an AI-powered fraud detection system developed using Machine Learning to identify fraudulent banking transactions.

The system uses XGBoost and SMOTE for handling imbalanced data and provides real-time fraud predictions using FastAPI and Streamlit.

---

## Features

- Fraud Detection using Machine Learning
- XGBoost Classifier
- SMOTE for imbalance handling
- FastAPI Backend
- Streamlit Dashboard
- Docker Support
- Real-time Prediction API
- Industry-level project structure

---

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- FastAPI
- Streamlit
- Docker
